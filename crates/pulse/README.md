# Pulse

## Description
Pulse is a voice node for Harmony, built on WebTransport. It seamlessly connects with the Harmony core server using Redis. Pulse handles all voice traffic, which is end-to-end encrypted with the help of MLS, and relays it between clients without the need for a complex WebRTC SFU. By leveraging WebTransport, Pulse can provide a performant voice communication experience with less moving parts than traditional WebRTC-based solutions.
